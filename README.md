# readme1

Instructivo paso a paso de como ejecutar el proyecto del backend

Abrir en Visual Studio Code los archivos Frontend y Backend.
El Backend se encuentran en: https://github.com/Nachoofiore/BackendGrupoJaP
El Frontend se encuentra en: https://github.com/Nachoofiore/GrupoJaP4 
En el archivo Backend seleccionar la terminal Git Bash e ingresar “npm run dev”.
Abrir el Frontend en la web.
Ingresar al eCommerce, en este caso “Soluginama”, con uno de los usuarios habilitados.
Se encuentran en la carpeta OUTH dentro del Backend.
